====================================
wbflip - flip the whitebox attribute
====================================

.. raw:: latex

    \begin{comment}

:code:`yosys> help wbflip`
--------------------------------------------------------------------------------

.. container:: cmdref


    .. code:: yoscrypt

        wbflip [selection]

    ::

        Flip the whitebox attribute on selected cells. I.e. if it's set, unset it, and
        vice-versa. Blackbox cells are not effected by this command.
