================================================
autoname - automatically assign names to objects
================================================

.. raw:: latex

    \begin{comment}

.. cmd:def:: autoname
    :title: automatically assign names to objects



    .. code:: yoscrypt

        autoname [selection]

    ::

        Assign auto-generated public names to objects with private names (the ones
        with $-prefix).
