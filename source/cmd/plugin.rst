=====================================
plugin - load and list loaded plugins
=====================================

.. raw:: latex

    \begin{comment}

.. cmd:def:: plugin
    :title: load and list loaded plugins



    .. code:: yoscrypt

        plugin [options]

    ::

        Load and list loaded plugins.


    .. code:: yoscrypt

        -i <plugin_filename>

    ::

            Load (install) the specified plugin.


    .. code:: yoscrypt

        -a <alias_name>

    ::

            Register the specified alias name for the loaded plugin


    .. code:: yoscrypt

        -l

    ::

            List loaded plugins

.. raw:: latex

    \end{comment}
